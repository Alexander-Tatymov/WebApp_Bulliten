Проект на C# Доска_для_заметок (Notes App): Сохранение, редактирование и удаление текстовых заметок.

Основные компоненты проекта
 Модель заметки (Note): содержит заголовок, текст, список тегов, дату создания и обновления
Контроллер (Notes Controller): управляет CRUD-операциями.
Видовые шаблоны (Views): формы для добавления/редактирования, список заметок.
Хранилище данных: использование встроенный механизм хранилище, EF Core (SQLite)

Технологии и Инструменты
Язык: C#
Фреймворк: ASP.NET Core
IDE: Visual Studio
База Данных: SQLite
ORM (Object-Relational Mapper):
Entity Framework Core (EF Core)
UI Framework: Razor Pages
